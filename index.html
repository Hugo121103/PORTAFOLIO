Detalles

“Calate” es como fue llamado el proyecto. Consiste en un robot autómata el cual debe ser programado con la finalidad de sacar a su contrincante de un área determinada. EL robot debe ser capaz de tomar decisiones por sí solo y dependiendo de las condiciones en las que se encuentre.
Se diseño su chasis, ruedas, circuitos y código desde cero. Utilizando así muchos componentes electrónicos, así como el entorno de programación Arduino, el cual está basado en C++.

¡Este autómata fue campeón en una competencia organizada por una universidad tecnológica!

Detalles

El presente brazo robótico fue impreso con material de plástico, consta de varias intersecciones las cuales permiten su movimiento, para ejecutar alguna acción se utilizan 3 servomotores los cuales tienen que ser programados grado por grado para poder realizar la acción requerida.
Se utilizo la placa Arduino como principal microprocesador y se desarrolló del código desde cero para poder ejecutar rutunas.
Así también se adapto un joystick y se creó una app móvil con comunicación bluetooth para poder controlar su movimiento de manera remota.
